### Content Insights
- since child themes are separate from parent themes and are simply overrides, the parent theme is free to update without causing conflicts or errors - unless the targetted classes are changed of course
- you can get equivalent functionality of grandchild themes by using the user customisations overrides (maybe you could use this for something like font preferences that then apply across all your sites)
- functions.php can be placed in the child theme and allows you to add functions to theme that otherwise would not be there and is often used to load the child theme style.css where the parent theme has a unique enqueue
